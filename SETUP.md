# Setup Instructions

All environment variables are configured in Vercel.
https://vercel.com/srikanth-2237s-projects/v0-job-seeker-platform/settings/environment-variables

## Required Variables

- NEXTAUTH_URL
- NEXTAUTH_SECRET
- GOOGLE_CLIENT_ID
- GOOGLE_CLIENT_SECRET
- LINKEDIN_CLIENT_ID
- LINKEDIN_CLIENT_SECRET
- RESEND_API_KEY
- STRIPE_SECRET_KEY
- STRIPE_PUBLISHABLE_KEY
- NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY
- DATABASE_URL
